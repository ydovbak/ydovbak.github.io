Android Application "TrailsApp"

In the TrailsApp a user can login using his Google account and create, plan and save trails or walks by dropping pins on the map. All the trails that were created by user are saved in the FireStore database and are connected to the users. Therefore as long as the user will login with the same account, his trails will be saved.
