Platformer game

You can watch this trailer video to see more about the game

Platformer game was "Industrial" was developed in GameMaker software program as part of the Game Development module.

The main challenge of this project was to simulate gravity and perform a good collision management as the main hero was stucking in walls and moving platforms.

"Industrial" consists of 10 different levels. Each level has different background image and background music. All events in the game are accompanied by sound effects which makes game more entertaining. A lot of powerups and enemies are all contributing to overall quality of the game.
