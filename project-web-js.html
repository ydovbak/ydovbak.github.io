Web Development JavaScript Project

In the second semester I designed this website to demonstrate my skills and knowledge in basic Javascript, HTML and CSS. The website consists of 5 pages:

  1. 'Home' page with a general overview of the website

  2. 'Test your basic math' page

  3. 'Unit Conversions' page where you can convert different units of mass, temperature and length

  4. 'Calorie Calculator' page where you can count daily calories needs

  5. 'Vectors Calculator' page for calculating sum, dot product, cross product and angle between two Vectors
